# Behigen

## Local development

- `npm install`
- `npm run dev`
- `npm run test`
- `npm run test:e2e`
- `npm run build`

## Content editing

- Products live in `src/content/products`
- Blog posts live in `src/content/blog`
- FAQ groups live in `src/content/faq`
- Tool configs live in `src/content/tools`
